“Easy program, affordable”

Overall: The program is very affordable--changed from another program to
this one and have found no problems with the calculations. The program is
mostly user friendly. I don't find it to be put together logically for me.I feel
like I must go through a lot of gyrations to get to the report that I need.
Sometimes things are promised to be completed in the applications that aren't
delivered when they are promised. There is sometimes a problem with response
--promises to call and then we have to make the call again because there is
no response. When you do call, the people are all friendly and try to be helpful.
